# Pixel ARTillery 2D

---

A small library for rendering Pixel Art with OpenGL 4.1 based on LWJGL 3.

## Requirements
|        | Version |
|--------|---------|
| JDK    | 21      |
| Maven  | 3.9.6   |
| OpenGL | 4.1     |
